Backup & Recovery

Stella Pay information is never included in backups or imports of data.

You can back up Tabs, update settings and search preferences to be used on another device or on this device in the future. You will receive a URL at the end of the backup allowing you to paste it onto any other device. The URL is a unique identifier that should not be shared if you have personal data stored in this session.
Backups only last 30 days like all other data stored on this session.
Restoring from a backup will always erase any existing data.
It is recommended to only back up your data if you need to put it onto another device, erase this browser or using a different version of Stella.
